<blockquote>
<p>Recreated an Instagram-style web application that allowed you to rate and comment on various dog pictures</p>
</blockquote>
<hr></hr>
  <h3><a href="https://we-heart-dogs.netlify.com/">Check it out on Netlify!</a></h3>
<hr></hr>
<p><strong>Built With:</strong></p>
<ul>
  <li>Vanilla JavaScript</li>
  <li>HTML</li>
  <li>Vanilla CSS</li>
  <li>Ruby on Rails</li>
  <li>Twitter API</li>
  <li>Twurl</li>
</ul>
<hr>
</hr>
<h3>Features</h3>
<ul>
  <li>Sort by:
    <ul>
      <li>Most Popular Dogs</li>
      <li>Highest Rated Dogs</li>
      <li>Most Commented on Dogs</li>
    </ul>
    </li>
    <li>"Heart" a Dog</li>
    <li>Rate a Dog</li>
    <li>Comment on a Dog</li>
</ul>
